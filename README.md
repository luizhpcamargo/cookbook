#COOKBOOK

Projeto utilizando rails 5.1.5 e ruby 2.3.1

Para rodar o projeto precisa somente instalar as gems e iniciar o server
`$ bundle install`
`$ rails s`

Para rodar os testes

`$ rspec spec`
